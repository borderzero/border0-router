WAN Configuration

WAN is a logical zone, not a bridge. Tick the interfaces that belong to the zone, set each one's addressing (DHCP or static), and pick the single active uplink that carries the live default route. Everything else stays defined but down. WiFi client interfaces are managed on the WiFi page; they appear here and can be chosen as the active uplink.
